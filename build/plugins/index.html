API Reference¶

Blocks has several APIs your plugin can use to interact with the system. The APIs are provided by the system services:

  • AccountService (blx()->account)
  • BlockTypesService (blx()->blockTypes)
  • ConfigService (blx()->config)
  • EmailService (blx()->email)
  • EntriesService (blx()->entries)
  • GlobalsService (blx()->globals)
  • PagesService (blx()->pages)
  • PathService (blx()->path)
  • HttpRequestService (blx()->request)
  • LocalizationService (blx()->i18n)
  • PluginsService (blx()->plugins)
  • TemplatesService (blx()->templates)
  • UsersService (blx()->users)
