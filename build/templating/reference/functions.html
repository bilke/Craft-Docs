Template FunctionsÂ¶

On top of the template functions that Twig comes with, Blocks provides a few of its own:

url

Returns a URL relative to your Site URL.

<a href="{{ url('company/contact') }}">Contact Us</a>
round

Rounds off a number to the closest integer.

{{ round(42.1) }} => 42
{{ round(42.9) }} => 43
ceil

Rounds of a number up.

{{ ceil(42.1) }} => 43
floor

Rounds a number down.

{{ floor(42.9) }} => 42
min

Returns the smallest number of a given set.

{{ min(1, 2, 3) }} => 1
max

Returns the largest number of a given set.

{{ max(1, 2, 3) }} => 3
getHeadNodes

Outputs all of the HTML nodes that have been queued up by the includeCssFile, includeCss, and includeHiResCss tags.

<head>
    <title>{{ siteName }}</title>
    {{ getHeadNodes() }}
</head>
getFootNodes

Outputs all of the HTML nodes that have been queued up by the includeJsFile and includeJs tags. It should be placed right before your </body> tag.

<body>
    <h1>{{ page.name }}</h1>
    {{ page.body }}

    {{ getFootNodes() }}
</body>
