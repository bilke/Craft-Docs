Template TagsÂ¶

On top of the template tags that Twig comes with, Blocks provides a few of its own.

redirect

Redirects the browser to the passed-in URL or site path.

{% if not user %}
        {% redirect 'login' %}
{% endif %}
exit

Cancels the output of the current template, and throws an HttpException, optionally with the status code you pass in.

{% set entry = blx.entries.id(entryId) %}

{% if not entry %}
        {% exit 404 %}
{% endif %}
includeCssFile

Queues up a <link rel="stylesheet"> tag with the passed-in CSS file URL, for inclusion via the getHeadNodes() function.

{% includeCssFile "/resources/css/global.css" %}
includeJsFile

Queues up a <script type="javascript"> tag with the passed-in JS file URL, for inclusion via the getFootNodes() function.

{% includeJsFile "/resources/js/global.js" %}
includeCss

Queues up a <style type="text/css"> tag with the passed-in CSS, for inclusion via the getHeadNodes() function.

{% set myCss %}
        .error { color: #f00; }
{% endset %}

{% includeCs myCss %}
includeJs

Queues up a <script type="javascript"> tag with the passed-in JS, for inclusion via the getFootNodes() function.

{% set myJs %}
        $('.date').datepicker();
{% endset %}

{% includeJs myJs %}
