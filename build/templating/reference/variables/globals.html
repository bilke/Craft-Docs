Global Variables¶

Blocks makes the following variables available to all templates:

blx

Most of Blocks’ template variables are tucked away in blx:

  • blx.entries – used to access your site’s entries
  • blx.request – used to access information about the current request
  • blx.users – used to access your site’s users
now

A date variable set to the current date and time.

Today is {{ now.format('M j, Y') }}.
siteName

The name of your site, as defined in Settings > General.

<h1>{{ siteName }}</h1>
siteUrl

The URL of your site, as defined in Settings > General.

<link rel="home" href="{{ siteUrl }}">
user

A user variable set to the currently logged-in user (if there is one).

{% if user %}
   Welcome, {{ user.friendlyName }}!
{% endif %}
globals

The location of any Global Blocks in the system.

<meta name="author" content="{{ globals.author }}">
<meta name="description" content="{{ globals.description }}">
